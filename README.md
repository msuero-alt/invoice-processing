# Invoice-Processing

This workbook demonstrates a typical Accounts Receivable workflow, including:

- **Invoices** – tracking invoice IDs, customers, dates, amounts, and payment status  
- **Payments** – reconciling payments with invoices using Excel formulas  
- **Summary** – pivot tables highlighting total invoiced, outstanding balances, and invoice counts per customer  

It’s designed to showcase practical Excel skills relevant to AR reporting, including VLOOKUP/XLOOKUP, SUM/IF formulas, pivot tables, and data filtering.  

This sample shows how raw invoice and payment data can be organized, analyzed, and summarized efficiently for reporting purposes.  

Download and open the raw file, `Accounts_Receivable_Workflow.xlsx`, to see the full workbook.
